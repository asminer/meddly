Obtaining MEDDLY

The source code for MEDDLY may be obtained by downloading a compressed tarball for a specific release, or by accessing the git repository directly.

Downloading

Source tarballs may be downloaded through SourceForge.

Subversion

The latest source code may be checked out anonymously from the git repository using:

git clone git://git.code.sf.net/p/meddly/code-git Meddly

It is also possible to browse the repository.

Developers should check out using:

git clone ssh://uname@git.code.sf.net/p/meddly/code-git Meddly

or

git clone https://uname@git.code.sf.net/p/meddly/code-git Meddly

where uname is your SourceForge username.
